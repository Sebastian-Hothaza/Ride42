# Ride42 App

The Ride42 App is the frontend interface for the Ride42 Trackday Management System, built with **React**. It allows users and administrators to view, register, and manage trackday events, motorcycles, and check-ins with a seamless and responsive experience.

▶️ [Try the live demo](https://demo.ride42.ca)

---

## Features

- **User Management:** View and manage registered participants.
- **Trackday Dashboard:** Browse available trackdays, register participants, and view schedules.
- **Motorcycle Garage:** Add, edit, and remove motorcycles associated with users.
- **QR Code Check-in:** Scan or assign QR codes for participants and motorcycles to streamline event check-in.
- **Payment Tracking:** View payment status and process payments securely via Stripe.

---

## Tech Stack

- **Frontend:** React
- **UI Components:** Modular, responsive React components
- **Integration:** Fully connected to the Ride42 API for backend operations

---

## Notes

- The app interacts with the Ride42 backend for all functionality.
- Designed for admins and participants alike for smooth trackday management.
- Fully responsive for desktop and mobile use.
